ST5230 Project — 复现步骤
==========================
最后更新: 2026-04-02

研究问题: LLM 的 uncertainty estimates 对语义等价的 prompt 变体有多敏感？

模型（最终选定）:
  1. mistralai/ministral-3b-2512   (开源, 3B)
  2. meta-llama/llama-3.1-8b-instruct  (开源, 8B)
  3. openai/gpt-4o                 (闭源)

数据集: SciQ (250 samples), TruthfulQA (250 samples)

模型替换说明: 见 docs/model_changes.txt

---
前置条件
---
  - Python 3.10+
  - OpenRouter API Key (存放在项目根目录 .env 文件中)

  .env 文件格式:
    OPENROUTER_API_KEY=sk-or-v1-xxxxxxxx

---
Step 1 — 安装依赖
---
  cd ST5230-Project
  pip install -r requirements.txt

  最小依赖（只用 API，不跑本地模型）:
    pip install openai python-dotenv numpy pandas scipy matplotlib

---
Step 2 — 准备数据（只需运行一次）
---
  python sources/data_utils.py

  从 HuggingFace 下载 SciQ 和 TruthfulQA，各采样 250 条，保存到:
    outputs/data/sciq_sample.json
    outputs/data/truthfulqa_sample.json

  注: 如果这两个文件已存在则会直接复用，不会重新采样。

---
Step 3 — 运行实验
---
  每条命令对应一个 (模型 × 数据集) 组合，共 6 次。
  所有模型通过 OpenRouter API 调用，无需 GPU。

  # Ministral-3B (开源小模型, 3B)
  python run_experiment.py --model mistralai/ministral-3b-2512 --dataset sciq --openrouter
  python run_experiment.py --model mistralai/ministral-3b-2512 --dataset truthfulqa --openrouter

  # Llama-3.1-8B (开源中等模型, 8B)
  python run_experiment.py --model meta-llama/llama-3.1-8b-instruct --dataset sciq --openrouter
  python run_experiment.py --model meta-llama/llama-3.1-8b-instruct --dataset truthfulqa --openrouter

  # GPT-4o (闭源大模型)
  python run_experiment.py --model openai/gpt-4o --dataset sciq --openrouter
  python run_experiment.py --model openai/gpt-4o --dataset truthfulqa --openrouter

  快速测试（先跑 3 条验证流程）:
    加 --max_samples 3

  断点续跑:
    中断后直接重新执行同一条命令，已处理的样本会自动跳过。
    如需重新开始: 加 --no_resume

  结果保存到 outputs/results/<model_tag>/ 目录:
    - <dataset>.jsonl        每行一条样本的完整推理结果
    - <dataset>_summary.json  汇总统计

---
Step 4 — 分析结果
---
  python analyze_results.py

  输出:
    终端打印 Table 1 汇总表（Flip Rate, LM Variance, Accuracy, 相关性等）

    outputs/figures/ 下生成 5 张 PDF 图表:
      fig1_main_results.pdf       — Flip Rate + LM Variance 跨模型对比
      fig2_lmvar_vs_flip.pdf      — LM Variance vs Flip 状态 (box plot)
      fig3_variant_accuracy.pdf   — 各 Prompt Variant 的准确率
      fig4_lmvar_distribution.pdf — LM Variance 分布直方图
      fig5_hidden_instability.pdf — Hidden Instability 散点图

---
Prompt Variants 说明
---
  每个样本使用 4 种语义等价的 prompt 变体:

  V0 (Baseline)      指令在前, A) 格式, "Answer:"
  V1 (Reworded)      不同措辞, A. 格式, "Your answer:"
  V2 (Question-first) 问题在前, (A) 格式, "Correct option:"
  V3 (Minimal)       无指令句, A: 格式, "Answer:"

  详见 sources/prompt_variants.py

---
关键指标
---
  1. Prediction Flip Rate   预测在 prompt 变体间翻转的比例
  2. Logit Margin Variance  top-1 与 top-2 logit 差值的方差 (衡量 confidence 波动)
  3. Hidden Instability     预测不变但 LM Variance > 5.0 的样本 (隐藏的不稳定性)
  4. Per-Variant Accuracy   每种 prompt 变体下的准确率

  注: USS (Uncertainty Stability Score) 和 Entropy 在汇总数据中也有计算，
  但由于 API 模型 top_logprobs 限制导致 floor 效应，在报告中以
  Logit Margin Variance 作为主要 uncertainty stability 指标。
